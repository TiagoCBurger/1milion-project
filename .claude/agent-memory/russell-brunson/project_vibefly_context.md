VibeFly is a marketing infrastructure company connecting AI tools to marketing platforms via MCP (Model Context Protocol).

**Value Ladder (as designed):**
1. Onerepo Business — template + lessons (entry product, one-time payment) — MVP status
2. MCP Meta Ads — live Meta Ads integration (future)
3. MCP Google — Google Ads + Analytics integration (future)
4. MCP Hotmart / Kiwify — sales data integration (future)
5. Onerepo para Agencias — multi-client version (future)
6. Comunidade — after audience is established (future)
7. Modules por nicho — niche-specific skills and structures (future)

**Entry product detail (Onerepo Business):**
- Git repo template as single source of truth for the business
- Includes: CLAUDE.md hierarchy, .rules/, .skills/ (create-post, campaign-brief, write-email, weekly-report), brand/, strategy/, campaigns/, clients/, metrics/, email-marketing/, integrations/
- Bonuses: AI image/video generation integration (fal.ai, kie.ai), free funnel publishing (GitHub Pages), email marketing without monthly tool fees (Resend/SES), and reseller/affiliate angle (Bonus 3 — "sell your own Onerepo")
- Bonus 3 is the network effect mechanism — buyers can become resellers for their own niches

**Brand identity:**
- Archetype: Mago (Magician) + Cara Comum (Everyman)
- Slogan: "Superpoderes de Marketing para voce e seus agentes de IA"
- Brand promise: "O VibeFly da a voce o time que voce nao tem"
- Tone: direct, confident, empathetic, irreverent — never corporate buzzwords

**Launch status (as of 2026-04-04):** Zero audience. Building from scratch.

**Why:** Tiago is highly technical but not a communicator, so the go-to-market strategy must work around content creation friction.
**How to apply:** Prioritize strategies that leverage technical proof, community seeding, and the Dream 100 approach over traditional content marketing or personal brand content creation.
