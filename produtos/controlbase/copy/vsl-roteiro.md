## NOTAS DE PRODUÇÃO

- Sem rosto. Screen recording puro converte bem.
- Microfone decente — áudio ruim mata conversão.
- Mostre o terminal real, o Claude Code real, arquivos reais — não exemplos fictícios.
- Ao abrir o CLAUDE.md e os arquivos de cliente, verbalize que são arquivos de texto, não código — reforça que qualquer um consegue fazer.
- Edite os erros de digitação e pausas longas, mas mantenha o tom humano.
- Adicione legendas — grande parte assiste sem som.
- Thumbnail: estrutura de pastas no terminal + headline "Você não precisa saber programar para usar o Claude Code."
